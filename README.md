# Simple AI Classroom Chatbot Practice Project

This is a basic practice assignment to explore building a local chatbot assistant connected to the Gemini API. 

## What it does
- Houses a modular code structure to communicate with Google's generative models securely.
- Serves a simple UI window where users can toggle between a classroom helper mode and a code explainer window.
- Handles conversation processing dynamically from a clean backend service layer.

## Setup & Running
1. Install requirements: `pip install -r requirements.txt`
2. Run the application layer: `python app.py`
3. Access locally: Open `http://127.0.0.1:5000` in any web browser.
